## Introduction
This repository contains several DSA questions solved in C. 

## Requirements
- GCC Compiler (or any C compiler of choice)
- Terminal

## How to Compile
Open a terminal in the project directory and run:
```bash
gcc prog_<q>.c -o prog
```
Replace `prog_<q>.c` with the name of the corresponding question's C source file.

## How to Run
After successful compile, execute:
```bash
./prog
```

## Example
```bash
gcc prog_1.c -o prog
./prog
```

## Notes
- Make sure GCC is installed via `gcc --version`
- On Windows, it is better to use:
```bash
program.exe
```
